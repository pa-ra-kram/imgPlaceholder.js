imgPlaceholder.js

A simple javascript plugin to create dynamic placeholder images

View on GitHub

Welcome to imgPlaceholder.

This is a very simple and light weight plugin which helps in generating placeholder images without using any external service.

The plugin is independent of any other external library, and has a jQuery version as well. Both the versions works exactly the same.

  • imgPlaceholder.min.js - No dependency (1.36 KB minified)

  • imgPlaceholder.jquery.min.js - jQuery required (1.26 KB minified)

Usage

Preferred method: Use 'data-src' attribute:

<img data-src="imgholder.js/320x24">

Simple 'src' attribute also works, but produces an error in console:

<img src="imgholder.js/320x24">

Various parameters can be passed in the following format

<img src="imgholder.js/360x240?color=#ffffff&bg=#000000&text=Hello">

The parameters can be passed

  • Default: 

    <img data-src="imgholder.js/200x100">

    Output:

  • Width only: 

    <img data-src="imgholder.js/200">

    Output:

  • Custom text: 

    <img data-src="imgholder.js/300x200?text=Hello World!!">

    Output:

  • Custom text color: 

    <img data-src="imgholder.js/300x200?color=#dd1233">

    Output:

  • Custom background color: 

    <img data-src="imgholder.js/300x200?bg=#F8DC3C">

    Output:

  • Custom font: 

    <img data-src="imgholder.js/320x240?font=Courier">

    Output:

  • Custom font size: 

    <img data-src="imgholder.js/320x240?size=28">

    Output:

Above paramters can be used in any combination

  • Combination 1: 

    <img data-src="imgholder.js/280x190?size=18&color=#fff&bg=#000&text=Combination 1">

    Output:

  • Combination 2: 

    <img data-src="imgholder.js/320x280?font=Helvetica&size=25&bg=#F0EFFC&color=#DD0000&text=Combination2">

    Output:

Supported parameters

  • text : Changes the text inside the image. Default text is width x height. Eg: 300x200.
  • color : Changes the text color inside the image. Accepts valid 3/6 digit hexcode color. Default color is #777.
  • bg : Changes the background color of the image. Accepts valid 3/6 digit hexcode color. Default color is #CCC.
  • font : Changes the text font inside the image. Default font is Arial.

Note: This plugin cannot be used as a background image yet.
